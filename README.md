## ğŸ Getting Started

### 1. Prerequisites
- Node.js (v18+)
- Python (v3.10+)

### 2. Environment Setup
Create a `.env` file in the root directory:
```env
GROQ_API_KEY="your_groq_key"
OPENWEATHER_API_KEY="your_weather_key"
```

### 3. Launching the Lab
**Install Dependencies:**
```bash
npm install
pip install -r requirements.txt
```

**Run Development Mode:**
```bash
npm run dev
```
